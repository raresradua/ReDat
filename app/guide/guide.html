ReDat Guide

Introducere

Aplicatia noastra contine paginile SignIn, Monitor, API, dintre care vom prezenta cum poate fi folosita SignIn si Montior.

Logare User

Odata intrat pe aplicatie user-ul va apasa pe butonul de SignIn din dreapta paginii, se va loga cu contul sau de Reddit si va fi redirected catre pagina Monitor.

Dupa logare

Folosind bara de navigatie, user-ul poate introduce de la tastatura sau din lista dropdown un subreddit la care este el abonat. Sau daca doreste parasirea aplicatiei, poate apasa pe butonul de LogOut.

Statistici

Ales un subreddit, se vor cauta in baza de date informatii legate de acel subreddit, daca nu sunt se va face request la API ul de la Reddit si vor fi generate pe pagina statistici aferente.

Plot-uri

Sunt generate plot-uri care vor putea fi downloadate fie in format SVG, colt dreapta sus este un buton de Download as SVG, sau in format CSV, sunt 3 butoane, cate unul pentru fiecare plot.

API

Utilizatorul poate folosi bara URL pentru a apela API ul care poate genera in format JSON informatii despre un subreddit prin /api/r/subreddit_nume sau poate genera informatii despre activitatea unui utilizator dintr-un anumit subreddit /api/r/subreddit_nume/username
